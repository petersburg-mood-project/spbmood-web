# spbmood-web
Веб-сервис социального картирования: центральный объект - размеченные карты города с разделением на маркеры социального самочувствия, а также социальные сферы с их соотнесением c эмоциямb, которые испытывают горожане.

## Анализ тональности и тематическое моделирование

Для анализа текстов из социальных сетей была разработана ансамблевая модель, включающая следующие компоненты:

- **Word2Vec** — для выделения семантических связей между словами.
- **BERT (Blanchefort/RuBERT-base-cased-sentiment)** — для классификации общего тона текста.
- **ruGPT-3** — для определения вероятностей принадлежности текста к базовым классам эмоций.

Модель достигла высокой точности классификации эмоционального фона текстов:

- **AUC**: 0.9512
- **F1**: 0.9362

На основе разработанной **модели базовых эмоций** текстам из социальных сетей присваивались эмоциональные индексы. Эти данные сопоставлялись с результатами социологических опросов, обеспечивая комплексный анализ социального самочувствия.

### Тематическое моделирование

Для тематического анализа использовался метод **LDA (Latent Dirichlet Allocation)**, адаптированный для коротких текстов из социальных сетей. Это позволило:

1. Соотнести тематические кластеры с конкретными социальными сферами.
2. Выявить ключевые темы обсуждений, связанные с городскими проблемами.
